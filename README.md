### 如何使用

---

在最新版本中，会有一个`config.json`，你可以通过更改它来更改博客名等等

如果你想写文章，那么可以更改`md`文件夹内的内容，不过还需要在`post`文件夹内添加一个和`md/洛谷P1001题解.md`差不多的文件，并且需要更改`config.json`的内容
